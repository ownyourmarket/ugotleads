# PromptExpert for UGotLeads — Chrome Extension

MV3 extension that inserts PromptExpert prompts and runs AI skills on chat pages
(ChatGPT, Claude, Gemini). Standalone package — not part of the app's pnpm workspace.

## Build

```bash
npm install
npm run build
```

This bundles `src/popup.js` and `src/content.js` into `dist/` via esbuild.

> Note: `src/popup.js` and `src/content.js` are **stubs** for this task (B1 — scaffold
> only). Real popup UI and content-script logic land in later tasks.

Use `npm run watch` during development for incremental rebuilds.

## Load in Chrome

1. Run the build (above) so `dist/` exists.
2. Open `chrome://extensions`.
3. Enable **Developer mode** (top right).
4. Click **Load unpacked** and select this `chrome-extension/` folder.

## Icons

`icons/16.png`, `icons/48.png`, `icons/128.png` are **temporary placeholders** —
solid teal (#14b8a6) squares generated by `icons/generate-icons.mjs` (pure Node,
no external deps). Swap for real branded icons before shipping.

## Config

`src/config.js` holds the public Firebase web config (the same `NEXT_PUBLIC_FIREBASE_*`
values the main app ships to every browser) plus `APP_URL`. These are not secrets.
